Security & Account Guides

Account Security Guide
Updated: August 15, 2023

Profile Settings

  • Avatar Update: Click the circular profile avatar area (100×100px). Supported formats: JPG, PNG, GIF (max 5MB). System auto-crops/resizes to 1:1 aspect ratio.
  • Personal Details: Full Name (required, 2-50 characters), Username (real-time availability checker, valid characters: a-z, 0-9, underscores, length: 3-20 characters).
  • Bio Editor: Character counter (max 500 characters) with live preview updates as you type.

Account Security

  • Password Requirements: 8+ characters, 1 uppercase, 1 lowercase, 1 number, 1 symbol (!@#$%^&*)
  • Two-Factor Authentication: Enable SMS authentication or authenticator app for enhanced security
  • Session Management: View active sessions, log out devices remotely

Troubleshooting & Support

Error Code Issue Resolution
SEC-101 Invalid password format Ensure password meets requirements: 8+ chars, uppercase, lowercase, number, symbol
SEC-203 Session conflict Clear browser cache and cookies, then log in again
SEC-305 Unauthorized deletion attempt Verify your identity and try again in 30 minutes
